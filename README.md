# Excel_Ecommerce_Dashboard
# This project analyzes eCommerce sales data using Excel. Various charts and visualizations have been created to gain insights into sales trends, customer behavior, and revenue patterns.About the Project
I worked on this project to analyze eCommerce sales data using Excel, leveraging various visualization techniques to extract meaningful insights. The objective was to identify sales trends, customer behavior, and revenue patterns to support data-driven decision-making.
#Tools & Techniques Used
Microsoft Excel: Primary tool for data processing, analysis, and visualization.
Pivot Tables: Used for data summarization and trend identification.
Charts & Graphs: Implemented bar charts, pie charts, scatter plots, and heatmaps.
Conditional Formatting: Highlighted key metrics and trends for better readability.
Useful Insights

Through this analysis, I was able to uncover the following key insights: 

Top-Selling Products & Categories: Identified the most profitable products and categories.
Seasonal Sales Trends: Observed peak sales periods and slow seasons.
Customer Demographics & Behavior: Analyzed customer preferences and purchasing habits.
Revenue Fluctuations: Compared revenue performance across different timeframes.
